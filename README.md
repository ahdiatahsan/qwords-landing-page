## Installation

Clone the project
```bash
https://github.com/ahdiatahsan/qwords-landing-page.git
```

Go to the project directory
```bash
cd qwords-landing-page
```

Install Dependencies
```bash
npm install
composer install
```

Configure .env
```bash
cp .env.example .env
```

Compiles and hot-reloads for development
```bash
npm run watch
php artisan serve
```
